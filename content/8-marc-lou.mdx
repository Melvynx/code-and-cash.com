Marc Louvion est le genre de hacker qui code une application en 1 jour, la publie le 2ème jour et génère ses premiers revenus le 3ème jour.

Sisi, il a créé pas loin de 16 applications en seulement 1 année.

Jusqu'à créer [shipfa.st](https://shipfa.st/), une boilerplate de code Next.js.

Tout ça alors qu'il n'a pas vraiment étudié le code à l'école.

On va voir :

- Son histoire et comment il en est arrivé ici
- Comment il a créé 16 applications en 1 an
- Comment vendre ton application
- Comment créer une personal brand sur Twitter
- Les clichés des développeurs DÉ-MON-TÉS.
- Et bien plus encore...
